# **CalpTek — Proyecto Formativo de Transformación Digital**

CalpTek es un proyecto formativo desarrollado en el marco del **Módulo de Proyecto Intermodular (2º CFMI, IES Ifach, Calp)**. Su objetivo es simular el funcionamiento de una empresa tecnológica dedicada a la digitalización, la infraestructura informática y el soporte técnico para pequeñas y medianas empresas (pymes).

Este repositorio contiene el **sitio web educativo**, la **documentación técnica**, y los **materiales de trabajo** creados por el equipo como parte del proceso de aprendizaje.

---

## 🚀 **Objetivos del proyecto**

- Comprender y simular el funcionamiento de una empresa de servicios tecnológicos.  
- Diseñar una infraestructura digital básica orientada a pymes.  
- Crear documentación técnica (SOP, políticas, procesos).  
- Desarrollar un sitio web informativo utilizando HTML y CSS.  
- Aplicar principios de sostenibilidad, ciberseguridad y buenas prácticas operativas.  
- Trabajar en equipo con roles definidos (operaciones, infraestructura, documentación).

---
